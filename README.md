## Acessando o banco de dados pelo terminal
**Comandos uteis:**

psql -U **_user_** -d **_db_**

psql --version

psql --help
        
**No banco de dados:**
- **listar os bancos:** \l
- **listar usuários:**  \du
- **listar tabelas:**   \dt
- **alterar bancos:**   \c nome_do_banco
- **informações da Conexão:** \conninfo
