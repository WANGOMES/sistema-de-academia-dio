## Como acessar o Banco de dados pelo terminal?
`Comandos uteis:`
        psql -U <user> -d <db>
        psql --version
        psql --help
        
No Banco de dados:
- listar os bancos:
        \l
- listar usuários:
        \du
- listar tabelas:
        \dt
- alterar bancos:
        \c nome_do_banco
- informações da Conexão:
        \conninfo
